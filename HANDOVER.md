## 技術メモ

### 画像について
- 既存HPの写真は直接URLで参照している（`http://www.touei-kk.co.jp/kenchiku/kenchiku.phot/〇〇.gif`）
- 画像はGIF形式が多い（古いHP由来）
- 画像が表示されない場合は既存HPのサーバーがダウンしている可能性あり → その際は写真を依頼主から再取得してGitHubにアップロードし、URLをローカルパスに変更する

### フォームの実装方法（Formspree使用の場合）
```html
<!-- index.html のフォームタグを以下のように変更 -->
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
  <!-- 各inputフィールドにname属性を追加 -->
  <input type="text" name="name" ...>
  <input type="email" name="email" ...>
  <input type="tel" name="tel" ...>
  <select name="category" ...>
  <textarea name="message" ...>
</form>
```

### GitHub Pages 構成
```
リポジトリ: yznok7714-ux/toe
└── files/
    ├── index.html        ← メインHP
    └── touei_works.html  ← 施工事例ページ
```
公開URL: `https://yznok7714-ux.github.io/toe/files/`（filesフォルダ内に配置のため）
※ ルートに配置すれば `https://yznok7714-ux.github.io/toe/` でアクセス可能

### リンクの依存関係
- `index.html` 内の「その他の施工事例を見る」ボタンは `touei_works.html` へのリンク
- 2ファイルは必ず同一ディレクトリに配置すること
- `touei_works.html` の「トップページへ戻る」リンクは `touei_hp_v3.html` を参照しているため、`index.html` にリネームした場合は修正が必要

## Claude Code への引継ぎ指示

このプロジェクトをClaude Codeで引き継ぐ際は、以下の順番で作業を進めることを推奨する。

1. `index.html`（旧 `touei_hp_v3.html`）と `touei_works.html` をローカルにクローンまたはダウンロードする
2. `touei_works.html` 内の「トップページへ戻る」リンクを `touei_hp_v3.html` → `index.html` に修正する
3. GitHub Pages での表示確認・デバッグを行う
4. メールアドレスが届き次第、Formspreeを使ってフォーム送信機能を実装する
5. 追加写真が届き次第、ギャラリーに追加する
